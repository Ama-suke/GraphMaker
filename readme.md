# 概要
csvファイルからグラフを作成するプログラムです。

※ 起動に結構時間がかかります

# 使い方
csvのデータは以下のように先頭にヘッダをつけてください。
| Header1 | Header2 | Header3 |
|---------|---------|---------|
| 23      | 56      | 0.0078  |
| 45      | 89      | 0.0012  |
| 67      | 34      | -0.0090 |
